### 📝 Project Overview
This gym logger app is a high-performance, mobile-first PWA designed for rapid tracking of workout sets. It features an integrated rest timer, automatic set counting per movement, and per-set technique notes.

**Key Features**:
- Real-time Firestore sync.
- Optimistic UI for no-latency logging.
- Advanced Autocomplete for exercises.
- Floating Rest Timer with customizable intervals.
