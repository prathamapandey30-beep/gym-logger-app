## 🚀 App Overview
The application is a premium, mobile-first gym workout logger designed for maximum entry speed and visual polish.

### Key Features:
- **Premium Glassmorphism UI**: Dark-themed, high-performance interface.
- **Rest Timer**: Floating interactive countdown to optimize recovery.
- **Set Counter**: Automatic detection of the current set number per exercise.
- **Movement Autocomplete**: Remembering your progress and exercises.
- **Robust Firestore Logic**: Error handling for database connectivity.
- **PWA Support**: Installable as a native app on home screens.
